# Legoable

Legoable 是构建可视化界面配置系统的，多种能力构成。

- 表层部分
  - 布局相关的约束（包括响应式、在各端上的体现）
  - 基础的数据输入项（各类表单组件）
  - 布局容器与导航
  - 多维度的数据展现（列表、树）
  - 各类图表
- 隐含的约束
  - 对于可访问性的指导，包括快捷键等
  - 操作记录的维持（表单的草稿，列表筛选项的缓存，自定义表头的缓存）
  - 统一的信息流
  - 默认的可访问性
